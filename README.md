# SerpentsHand
SCP: Classified Site plugin

Данный плагин добавляет в игру новый игровой класс - Длань Змея. Сам плагин эту роль не спавнит, но можно заспавнить её с помощью админпанели или плагина NewSupportManager.

Описание роли:
- **Команда**: SCP. Игнорирует все механики SCP (не может заагрить скромника, например, и т.д.);
- **Вещи в инвентаре**: взламывающая ключ-карта, ключ-карта специалиста по содержанию, АК-12 и патроны к нему, маленькая аптечка, флешка;
- **Модель**: перекрашенная в красный цвет модель Повстанца Хаоса;
- **Здоровье (по умолчанию)**: 120 хп. Это можно изменить в конфиге с помощью параметра **SH_health**.

Можно изменить иконку класса в админпанели на любую другую. Изображение должно быть квадратным  и размещено в папке *Plugins/client/[ПЛАТФОРМА]* (необходимо продублировать его для всех платформ). Все платформы написаны в документации по плагинам SCP: Classified Site.
